---
title: 'Get All Templates'
openapi: 'GET /templates/all'
---

## Overview
Retrieve a list of all available scribing templates. The <code>public_id</code> of the selected template will be required when submitting a scribing request.

## Query Parameters
- **`page`** (integer, optional) - Page number (default: 1).
- **`limit`** (integer, optional) - Number of results per page (default: 50).
